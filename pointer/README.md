## ✅ PointerPractice.c – 포인터 학습 예제

### ✨ 주요 학습 요소
- 변수 주소 출력 (`&변수`)
- 포인터 변수 선언 및 참조 (`*포인터`)
- 포인터를 활용한 값 수정
- 포인터와 배열의 관계
- 함수로 값 전달 vs 주소 전달 비교
- `scanf`에서 왜 `&변수`를 사용하는지 이해

### 📌 주요 함수

- `swap(int a, int b)`  
  → 값을 복사해서 전달하는 방식 (Call by Value)

- `swap_addr(int* a, int* b)`  
  → 변수의 주소를 전달하여 실제 값을 바꾸는 방식 (Call by Address)

- `changeArray(int* ptr)`  
  → 포인터를 이용한 배열 원소 값 변경

## 🎮 FishGame.c – 시간 기반 수분 관리 생존 게임

### 🐟 게임 설명

- 6개의 어항에 각각 물고기가 1마리씩 있습니다.
- 시간이 지날수록 어항의 물이 증발하며 줄어듭니다.
- 플레이어는 번호를 입력해 원하는 어항에 물을 줄 수 있습니다.
- 물고기의 생명은 어항 물 수치가 `0`보다 크냐에 따라 결정됩니다.
- **20초마다 레벨이 오르며**, 증발 속도는 더 빨라집니다.
- 레벨 5에 도달하면 **게임 클리어!**, 모든 물고기가 죽으면 **게임 오버!**

### 🕹️ 게임 진행 방식

1. 화면에 6개 어항의 상태가 출력됩니다.
2. 사용자 입력으로 1~6 중 어항을 골라 물을 줍니다.
3. 시간이 경과할수록 자동으로 물이 줄어듭니다.
4. 20초마다 레벨이 증가하며 난이도도 상승합니다.
5. 물이 0이 되면 해당 물고기는 죽습니다.
6. 모두 죽거나 레벨 5 달성 시 종료됩니다.

### ⏱️ 시간 기반 요소

- `clock()` 함수를 이용해 게임 시간 계산
- `preElapsedTime`을 활용해 물 증발량 계산
- 레벨 별 증발 속도 증가:  
  `arrayFish[i] -= level * 3 * elapsedTime`
