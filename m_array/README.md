# 🧠 C 배열 학습 및 기억력 게임 프로젝트

이 프로젝트는 C 언어의 **1차원, 2차원, 3차원 배열 구조**를 학습하고,  
이를 응용한 **간단한 동물 카드 짝 맞추기 게임**을 구현하는 데 목적이 있습니다.

---

## 📁 파일 구성

| 파일명               | 설명 |
|----------------------|------|
| `m_array.c`          | 배열 구조 학습용 예제 코드 (1차원~3차원 배열 출력) |
| `m_array_project.c`  | 동물 카드 짝 맞추기 게임 구현 (4x5 배열 기반) |
